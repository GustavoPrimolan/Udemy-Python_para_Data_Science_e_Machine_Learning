<h1>Configuração do ambiente de trabalho</h1>

* Usaremos a versão mais atualizada do Python através do ambiente Anaconda.

* Python package manager
* pip3 install jupyter ou pip install jupyter (depende a versao)

* Para novos usuários recomendado utilizar a instalação do Anaconda.

<h1>Jupyter Notebook</h1>

* Abrir o jupyter notebook
* prompt: jupyter nootebook

<h2>Configurando ambientes virtuais</h2>

* Ambientes virtuais permitem que você defina instalações personalizadas e virtuais do Python.
* Você pode ter multiplas versões do Python e de suas bibliotecas instaladas.
